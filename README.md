# MediMate Frontend

This repository contains the user interface for the MediMate Insurance Platform.

## Features
- Patient Dashboard for Claim Submission
- Hospital Dashboard for Document Management
- Dynamic Form Generation
- Mediclaim Form Generation workflows

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment variables mapping to `.env` based on `.env.example` if available.

3. Start the development server:
   ```bash
   npm start
   ```
   *(or `npm run dev` depending on the framework)*
